# Mapeo de agua
>Versión 1.0.0

Este proyecto surge de la necesidad de hacer un mapeo de los reportes de agua sucia a lo largo de diversas colonias en Jalisco.

El código recibe una base de datos con las ubicaciones de los reportes, los transforma a coordenadas y genera un mapeo visual de los datos.
